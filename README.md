#  Day 6 – Data Visualization Fundamentals with Matplotlib

## Overview

This project demonstrates the fundamentals of **data visualization** using **Python** and **Matplotlib**. The notebook explores the Iris dataset and recreates six essential chart types commonly used during **Exploratory Data Analysis (EDA)**.

Each visualization includes a brief interpretation to explain the insights the chart provides, helping build both technical and analytical skills.

---

## Objectives

* Learn the fundamentals of data visualization with Matplotlib.
* Understand when to use different chart types.
* Explore patterns, distributions, relationships, and correlations within a dataset.
* Practice presenting insights through clear and informative visualizations.

---

## Dataset

**Iris Dataset**

The project uses the built-in Iris dataset available through `scikit-learn`, containing measurements of iris flowers from three different species.

Features include:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width
* Species

---

## Visualizations Included

###  Line Chart

Displays how sepal length changes across the dataset samples.

**Insight:** Helps observe trends and fluctuations in numerical values over ordered observations.

---

###  Bar Chart

Shows the number of flowers belonging to each species.

**Insight:** Useful for comparing counts across categories.

---

###  Histogram

Visualizes the distribution of petal length values.

**Insight:** Reveals the frequency distribution and overall spread of the data.

---

###  Boxplot

Summarizes the distribution of petal width.

**Insight:** Highlights the median, quartiles, variability, and potential outliers.

---

###  Scatter Plot

Compares sepal length with petal length.

**Insight:** Helps identify relationships or correlations between two numerical variables.

---

###  Heatmap

Displays the correlation matrix of numerical features.

**Insight:** Makes it easy to identify strongly positive or negative correlations among variables.
